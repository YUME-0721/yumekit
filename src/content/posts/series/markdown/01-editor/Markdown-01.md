# Markdown 编辑器

Markdown 是一种轻量级的标记语言，通过简单的标记语法使文本具备一定的格式，它的目标就是实现易读、易写。要使用 Markdown 编写文档，你需要一个合适的 Markdown 编辑器。

## 推荐的 Markdown 编辑器

### 1. Visual Studio Code

Visual Studio Code（简称 VS Code）是微软开发的免费、开源的代码编辑器。通过安装 Markdown 相关扩展，可以获得强大的编辑和预览功能。

**特点：**
- 免费开源
- 功能强大，支持丰富的扩展
- 实时预览
- 语法高亮
- 支持多种主题
- 跨平台（Windows、macOS、Linux）

**安装方法：**
1. 访问 VS Code 官网下载安装包
2. 安装后，在扩展商店搜索 "Markdown" 相关插件
3. 推荐插件：Markdown All in One、Markdown Preview Enhanced

### 2. Typora

Typora 是一款所见即所得的 Markdown 编辑器，界面简洁，使用体验优秀。

**特点：**
- 所见即所得（WYSIWYG）
- 界面简洁美观
- 支持数学公式、表格、代码块
- 支持导出多种格式（HTML、PDF、Word 等）
- 跨平台

**注意：**
- Typora 现在是收费软件（有免费试用期）
- 需要访问官网下载：https://typora.io

### 3. Editor.md

Editor.md 是一个开源的在线 Markdown 编辑器，功能强大，支持丰富的扩展。

**特点：**
- 开源免费
- 支持实时预览
- 支持 TeX 科学公式（基于 KaTeX）
- 支持流程图（Flowchart）和时序图（Sequence Diagram）
- 支持代码高亮
- 支持图片上传和粘贴

**GitHub 地址：**
- https://github.com/pandao/editor.md

**在线测试地址：**
- https://pandao.github.io/editor.md/

### 4. 其他推荐编辑器

#### Atom
- GitHub 开发的免费编辑器
- 支持丰富的 Markdown 插件
- 已停止维护（不推荐新用户使用）

#### Sublime Text
- 轻量级编辑器
- 需要安装 Markdown 插件
- 性能优秀

#### Notion
- 集成了 Markdown 功能
- 支持团队协作
- 功能丰富但相对较重

## 如何选择编辑器

根据你的需求选择合适的编辑器：

- **初学者**：推荐 Typora，界面简洁，所见即所得，学习成本低
- **开发者**：推荐 VS Code，功能强大，可扩展性强
- **在线使用**：推荐 Editor.md，功能丰富，无需安装
- **免费需求**：推荐 VS Code 或 Editor.md
- **付费意愿**：Typora 是最佳选择

## Markdown 文件后缀

Markdown 编写的文档后缀通常为：
- `.md`
- `.markdown`

## 在线编辑器

除了本地安装的编辑器，你还可以使用在线 Markdown 编辑器：

- 菜鸟工具 Markdown 在线编辑器：https://c.runoob.com/front-end/712/
- StackEdit：https://stackedit.io/
- Dillinger：https://dillinger.io/

在线编辑器的优点是无需安装，随时随地使用，但需要网络连接。

## 总结

选择合适的 Markdown 编辑器可以大大提高写作效率。建议初学者从 Typora 开始，熟悉 Markdown 语法后，可以根据自己的需求切换到 VS Code 或其他编辑器。无论选择哪款编辑器，掌握 Markdown 语法都是最重要的。
