## 文章格式说明

### 基本语法

你可以使用 **粗体**、*斜体*、~~删除线~~ 和 `行内代码`。

### 列表

无序列表：
- 第一项
- 第二项
- 第三项

有序列表：
1. 步骤一
2. 步骤二
3. 步骤三

### 代码块

```javascript
// JavaScript 代码示例
function hello(name) {
    console.log(`Hello, ${name}!`);
}

hello('World');
```

```python
# Python 代码示例
def hello(name):
    print(f"Hello, {name}!")

hello("World")
```

## 关于系列文章

如果你想写系列文章，可以：

1. **使用相同的标签**：给同系列文章添加相同的标签，如 `[React教程, 前端]`
2. **文件命名规范**：使用 `series-01-intro.md`、`series-02-basics.md` 这样的命名
3. **在文章中互相引用**：在文章末尾添加"上一篇/下一篇"链接
